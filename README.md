# Backend (Node.js + Express)

Handles API requests, integrates with Gemini API, and curates startup analysis data.

## Setup

1. Install dependencies:
   ```
npm install
   ```
2. Create a `.env` file with your Gemini API key:
   ```
GEMINI_API_KEY=your-key-here
   ```
3. Start the server:
   ```
npm run dev
   ```
